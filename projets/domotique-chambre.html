Domotique & IoT : Chambre Connectée

Automatisation d'éclairage et station météo connectée

ESP32 IoT Cloud iOS Widget

1. Le Matériel utilisé

Pour ce projet, j'ai choisi l'ESP32 pour sa connectivité Wi-Fi native. J'ai couplé ce microcontrôleur à un servomoteur pour l'action physique et un capteur de précision (BME280) pour les relevés environnementaux.

Le microcontrôleur ESP32

Servomoteur pour l'actionneur

Capteur Température, Humidité et Pression

2. Conception Mécanique "Non-Invasive"

L'objectif était de pouvoir commander la lumière sans modifier l'installation électrique. J'ai conçu un boîtier en carton renforcé fixé au mur. À l'intérieur, le servomoteur est positionné de manière à actionner l'interrupteur mécaniquement via un bras de levier programmé.

Installation du servomoteur sur l'interrupteur

3. Arduino IoT Cloud & Dashboard

J'ai connecté l'ESP32 à l'Arduino IoT Cloud. Cela me permet d'avoir un tableau de bord (Dashboard) accessible partout dans le monde pour surveiller les données de ma chambre et piloter la lumière à distance via une interface web ou mobile.

Interface de contrôle Cloud

4. Serveur Web Local et Widget iPhone

En parallèle du Cloud, l'ESP32 héberge un serveur Web local ultra-léger. J'ai utilisé cette API locale pour créer un Widget sur mon iPhone. Je peux ainsi voir la température et l'humidité directement sur mon écran d'accueil sans même ouvrir d'application.
